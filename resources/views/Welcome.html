Getting started

Adding a route


// GET route.
$app->get(route, response);

// POST route.
$app->post(route, response);

The route must always start with a slash.

The response can be an anonymous function or an array specifying either, a View or a controller.


// Response is an anonymous function.
$app->get('/', function() {
    echo "Welcome!";
});

// Response is a controller.
$app->get('/', ['controller' => 'WelcomeController']);

/**
 * When the response is a view you can also give it a title and pass it some data.
 */
$app->get('/', [
    'view' => [
        'view' => 'Users',
        'title' => 'Users',
        'data' => ['name' => 'Alex']
    ]
]);

You can also create a route that responds to two URI's.


// Multi route
$app->get(['/', '/home'], response);
